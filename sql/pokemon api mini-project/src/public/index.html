available endpoints for the pokemon api

/pokemon

all pokemon in pokedex

↳ optional queries:

minHp, exclude, excludeType, types
/pokemon/:id

a single pokemon specified by id
/pokemon/essential

all pokemon in pokedex, name and type only
/pokemon/essential/:id

a single pokemon specified by id, name and type only
/pokemon/non-mythical

all pokemon in pokedex that are not mythical

↳ optional query:

minLevel
/pokemon/search

all pokemon in pokedex that match the query

↳ required query:

name
/pokemon/fire-or-electric

all pokemon in pokedex with a fire or electric type
/pokemon/level-range

all pokemon in pokedex within a specified level range

↳ required queries:

minLevel, maxLevel
/pokemon/sort

all pokemon in pokedex sorted by hp, attack, or level

↳ required query:

sortBy
/pokemon/leaderboard

top 5 pokemon in pokedex based on level
/pokemon/count

a count of all pokemon in pokedex
/pokemon/stats

a collection of stats about the pokemon in pokedex
/pokemon/types

a count of all pokemon in all given types

↳ optional query:

populous
