## Running the Projects 🎬

1. Clone the repository.

2. Ensure Node.js is installed on your computer.

3. Open a terminal in the desired TypeScript project directory.

4. Install dependencies:
    ```bash
    npm install
    ```

5. Run the project:
    ```bash
    npm run dev
    ```
